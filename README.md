# AI-spam-call-detector-using-AWS-and-Gemini 
A spam call detector with about 90 percent accuracy
the following detector leverages google gemini and aws transcribe to acheive the goal  
  
  
To use the following spam call detector (type the following commands in folder terminal):  
1.for mac/linux:  
 ```source .venv/bin/activate```  
for windows:  
 ```.\venv\Scripts\activate```  
2.change directory to callnsight folder  
3.```python3 manage.py runserver```  
4.if package unavailable errors are shown download the package via the following command  
```pip install package_name```  
  
Note: Add gemini and aws api key for it to work and still Working on the loading page but the spam call detection still works
